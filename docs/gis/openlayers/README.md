## 介绍

Openlayers是一个专为WebGIS客户端开发提供的javascript类库包，用于实现标准格式发布的地图数据访问。因此，广大的WebGIS开发者借助Openlayers强大的拓展功能，开发出了各具特色的WebGIS应用系统。
[官网地址](https://openlayers.org/)
